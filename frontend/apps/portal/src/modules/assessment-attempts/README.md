# Assessment Attempts

Learner attempt frontend for published assessment snapshots.

- start or resume attempt from `/assessments/[assessmentId]/attempt`
- list learner-owned attempts at `/attempts`
- render immutable published snapshot questions at `/attempts/[attemptId]`
- save draft answers, submit attempts, and show result placeholders only

This module does not implement grading runtime, result release, code execution, or proctoring.

# Enterprise White-Labeling

## Scope

Enterprise supports branding controls, custom domains, and organization-level UI identity.

## Capabilities

- Custom logos, color tokens, and branded communication templates.
- Domain mapping and tenant routing via Cloudflare edge.
- Signed asset access controls for protected branded assets.

## Rules

- White-labeling is entitlement-gated.
- Branding settings are tenant-scoped and auditable.
- UI theming and asset paths must resolve through typed runtime config, not ad-hoc constants.

# API Contracts

Runtime API contracts and transport schemas for Go nodes.

Execution API status:

- No execution API is production-ready yet.
- Go execution node currently remains health-only placeholder infrastructure.
- No sandbox implementation exists yet.
- Execution providers may be external executors, container sandboxes, or managed services.
- Execution APIs must not be callable directly by learners.
- Execution nodes must never receive platform database credentials.
